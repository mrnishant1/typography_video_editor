# 🎬 Typography Video Editor

A browser-based subtitle and typography editor for creating animated captions and stylish text overlays for videos.

**🌐 Live Demo:** https://mrnishant1.github.io/typography_video_editor/


## Features

* 🎬 Everything is automitic, nothing to edit
* ✨ Beautiful subtitle styling
* 🎨 Custom fonts (including Hindi/Devanagari support)
* 🎯 Precise subtitle positioning
* 🎞️ Timeline-based subtitle 
* 📱 Responsive interface
* ⚡ Runs completely in the browser
* 🌍 No installation required



## Tech Stack

* HTML5 Canvas
* JavaScript

## Why I Built This

Most subtitle tools are either too limited or require expensive subscriptions. I wanted a fast, browser-based editor that gives creators full control over subtitle styling, typography, and animations without uploading their videos to a server.

This project is built with performance and flexibility in mind, making it suitable for YouTube videos, Shorts, Instagram Reels, and other social media content.
